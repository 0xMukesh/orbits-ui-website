## Usage

```jsx
<Avatar />
```

## Props

| Prop            | Type   | Default   | Description                                    |
| --------------- | ------ | --------- | ---------------------------------------------- |
| size            | number | "medium"  | The size of the avatar.                        |
| src             | string |           | The url of the image to be used as the avatar. |
| alt             | string |           | The alternative text for the avatar.           |
| avatarColor     | string | "#F1F6FC" | Color for the avatar face.                     |
| backgroundColor | string | "#6FC2FF" | Background color of the circle                 |
