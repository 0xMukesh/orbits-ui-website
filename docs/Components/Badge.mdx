## Props

| Prop  | Accepts                    | Type   | Default | Description                      |
| ----- | -------------------------- | ------ | ------- | -------------------------------- |
| size  | small, medium, large       | string | small   | The size of the badge component  |
| color | primary, secondary, danger | string | primary | The color of the badge component |
| value | any text                   | string | DEFAULT | The text for the badge component |
